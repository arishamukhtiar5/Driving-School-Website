# Driving School Website 🚗

This is a responsive **Driving School Website** designed to provide information about driving lessons, pricing plans, and services. The website is built using **HTML, CSS, and JavaScript** and is fully responsive for desktop and mobile devices.

## 🌐 Live Demo

You can view the live website here:
**Live Link:** https://arishamukhtiar5.github.io/Driving-School-Website/

## 📌 Features

* Responsive Navigation Bar
* Modern and Clean UI Design
* Smooth Scrolling Between Sections
* Mobile-Friendly Layout
* Contact Form Section
* Easy-to-Read Pricing Plans

## 📂 Sections Included

### 🏠 Index / Home

The homepage introduces the driving school with a welcoming banner, key highlights, and call-to-action buttons for users to explore services or contact the school.

### ℹ️ About

This section provides information about the driving school, its mission, experience, and commitment to safe and professional driving education.

### 💰 Pricing

Displays different driving lesson packages with clear pricing details so users can easily choose the plan that fits their needs.

### 🛠️ Services

Lists the services offered by the driving school, such as:

* Beginner Driving Lessons
* Advanced Driving Training
* Road Test Preparation
* Defensive Driving Courses

### 📞 Contact

Allows users to get in touch with the driving school through a contact form, phone number, email address, or location details.

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
